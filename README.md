# RefApp_Capolavoro25-26
RefApp è un'applicazione mobile Android che permette agli arbitri di calcio di salvare le loro partite, allenamenti, raduni, note ecc. ecc.
Inoltre, nella applicazione è possibile vedere degli aggiornamenti inviati tramite un piccolo server web.
L'applicazione permette inoltre di collegarsi al sistema Sinfonia4You tramite un visualizzatore web e salvare le credenziali crittografate localmente per poter accedere tramite pin o sistemi biometrici.
L'app invia delle notifiche un giorno prima e alcune ore prima dell'evento per evitare di dimenticarlo.
L'applicazione sfrutta un database locale nel quale vengono salvati tutti i dati, ed è stata interamente programmata in Dart/Flutter (al momento prevede solo Android, ma seguirà supporto per iOS).
Alcune parti dell'applicazione sono in corso di sviluppo.

# Installazione Android
Per installare l'app su Android per testare questa versione Beta è necessario scaricare il file "./files/app.apk" (versione Beta al momento e non definitiva) ed installare l'apk sul proprio dispositivo.
